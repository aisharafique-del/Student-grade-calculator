 Student Grade Calculator - C++ 

A console-based Student Grade Management System built in C++ demonstrating OOP concepts, file handling, and data processing for multiple students.

##  Features

- Add Multiple Students: Input and store details for N students
- OOP Implementation: Uses `Student` class with encapsulation
- Automatic Calculations: Total marks, percentage, average, grade assignment
- Grade System: A+, A, B, C, D, F based on percentage
- File Handling: Save all records to `students.txt` & read from file
- View Records: Display all students with full result summary
- Data Persistence: Records don't disappear after program closes
